# The Doll of Lolita

Text-based game "The Doll of Lolita" (Russian: "Кукла Лолиты") by Yuri Pavlenko (Russian: Юрий Павленко), a.k.a. Goraph.

The game is in Russian, developed for URQ_DOS, and adapted for [UrqW](https://github.com/urqw/UrqW) text-based game engine.

This repository contains the source code and other resources of the game. The game is available for launch in the [UrqW engine online catalog](https://urqw.github.io/UrqW/#doll_of_lolita).

The game was first published on July 6, 2008.

Interactive fiction identifier (IFID) of the game which is its MD5 hash: A9CE8E373FB4815E76BCEB6231526E5B

## Description

Я подошёл к своему дому. Моё сердце колотилось от предвкушения встречи с ней. Сегодня был её день рождения.
